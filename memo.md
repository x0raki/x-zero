# 補助メモ

このページは、グループチャットの ChatGPT に追加で見せたい前提情報を置くための公開メモです。

## 使い方

- チャットでこの URL を共有する
- 必要なら「この公開メモも参照してください」と伝える
- 友人にも見せられる公開前提で書く
- プライベートな情報は、公開されても問題ない範囲にとどめる
- 個人情報、機密情報、限定公開の URL は書かない

## このメモの役割

- ここは「追加で渡したい短い前提」だけを置く
- 長い本文や設定全文は別ファイルに分ける
- 公式のカスタム指示欄に入っている内容を、ここに重ねて全部は貼らない
- 重要なことはカスタム指示側、ここは補助的な内容に寄せる

## 現在の前提

- 公式のカスタム指示は別に設定済み
- この `memo.txt` は、必要な補足だけを軽く渡すために使う
- カスタム指示の全文を確認したいときは `custom-instructions.md` か `custom-instructions.txt` を見る

## 編集するときの目安

- まずはこの `memo.md` を編集する
- 編集後、同じ内容を `memo.txt` にも反映する
- GitHub ではファイルを開いて鉛筆マークからそのまま編集できる
- もし先に `memo.txt` を直したら、その新しい内容を `memo.md` に反映する
- 要するに、新しいほうを正として両方そろえる

## 補助メモの型

### 全体に関わること

- ここに全員共通の補足を書く
- 例: 会話の前提、最近の共有文脈、今はあまり触れなくてよい話題

### らきむぼん（僕）に関わること

- ここにらきむぼん向けの補足を書く
- 例: 最近の関心、話し方の補足、覚えておくと会話がつながりやすいこと

### 竜司（友人）に関わること

- ここに竜司まわりの補足を書く
- 例: 話題の前提、文脈、最近の共有事項

### メモ

- 重要なルールはカスタム指示側に寄せる
- ここには「追加で渡すと少し会話が良くなる補足」だけを書く
- 長くなりすぎるなら別ファイルに分ける

## 参照先

- 共有用: `/memo.txt`
- 編集用: `/memo.md`
- カスタム指示全文: `/custom-instructions.txt`
