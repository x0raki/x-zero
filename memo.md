# 補助メモ

このページは、グループチャットの ChatGPT に追加で見せたい前提情報を置くための公開メモです。

## 使い方

- チャットでこの URL を共有する
- 必要なら「この公開メモも参照してください」と伝える
- 友人にも見せられる公開前提で書く
- プライベートな情報は、公開されても問題ない範囲にとどめる
- 個人情報、機密情報、限定公開の URL は書かない

## 前提情報

- ここにプロジェクトや運用の前提を書く
- 例: 用語集、目的、避けたい表現、参考リンク

## 判断ルール

- ここに優先順位や判断基準を書く
- 例: まず最小変更、既存の命名に合わせる、提案は3案まで

## 参照先

- ここに公開ドキュメントの URL を並べる
